# 📊 Zudio Retail Performance Analysis Dashboard (Project V)

## 📌 Overview
This Excel dashboard analyzes Zudio retail sales data to evaluate performance, 
profitability, and product trends for better business insights and decision-making.

## 🎯 Objective
- Monitor overall sales, profit, and order performance  
- Analyze category-wise and product-wise contribution  
- Identify top-performing and low-performing products  
- Track monthly sales and profit trends  

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Slicers  
- Data Cleaning & KPI Design  

## 📈 Key Insights
- Identified top-performing products contributing to profit  
- Highlighted category-wise performance differences  
- Tracked monthly fluctuations in sales and profit  
- Analyzed regional performance and demand patterns
