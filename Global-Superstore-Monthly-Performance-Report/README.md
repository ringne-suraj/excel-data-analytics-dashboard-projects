# 📊 Global Superstore Monthly Performance Report (Project IV)

## 📌 Overview
This Excel dashboard analyzes global superstore sales data 
to evaluate monthly performance, profitability, and key business metrics for decision-making.

## 🎯 Objective
- Monitor current month sales performance  
- Compare growth with previous month  
- Analyze category-wise and region-wise performance  
- Identify low-performing areas and improvement opportunities  

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Slicers  
- Data Cleaning & KPI Design  

## 📈 Key Insights
- Tracked monthly sales growth and performance trends  
- Identified top-performing categories and regions  
- Highlighted low-performing states impacting profitability  
- Analyzed profit margin and overall business health
