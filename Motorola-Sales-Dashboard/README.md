📊 Motorola Sales Insights Dashboard (Project I)
📌 Overview

This Excel dashboard analyzes Motorola sales data to track performance, transactions, and key business metrics for informed decision-making.

🎯 Objective
Monitor overall sales performance
Analyze brand-wise and city-wise sales
Track transaction volume and pricing trends
Visualize monthly performance insights

🛠️ Tools Used
Microsoft Excel
Pivot Tables
Charts & Slicers
Data Cleaning & KPI Design

📈 Key Insights
Identified top-performing brands and cities
Tracked monthly sales decline and growth trends
Analyzed customer transaction patterns
Highlighted key revenue-driving segments
