📊 Customer Behavior Analysis Dashboard (Project II)

📌 Overview

This Excel dashboard analyzes customer behavior data to understand spending patterns,
engagement levels, and demographic insights for better business decisions.

🎯 Objective
Analyze customer demographics (age, gender, membership)
Track spending behavior across cities and segments
Measure customer satisfaction and engagement levels
Identify high-value and low-value customer groups.

🛠️ Tools Used
Microsoft Excel
Pivot Tables
Charts & Slicers
Data Cleaning & KPI Design

📈 Key Insights
Identified high-spending customer segments and cities
Analyzed satisfaction levels across different customer groups
Highlighted differences in purchasing behavior
Segmented customers based on engagement and activity
